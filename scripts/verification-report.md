## 1. Kindergarten Verification

### 1.1 Names — PARTIALLY REAL
- Kindergarten names follow real Lithuanian naming conventions (l-d, lopšelis-darželis)
- Many names match real Lithuanian kindergartens (Gintarėlis, Vieversys, Pagrandukas, etc.)
- **However**, some are assigned to wrong cities or duplicated across cities
- DB has 16 Alytus kindergartens; real Alytus has 16+ public kindergartens — names partially match

### 1.2 Addresses — MOSTLY FABRICATED
Cross-checked against official Alytus municipality data (tavovardas.com, alytus.lt):

| DB Entry | DB Address | Real Address | Match? |
|----------|-----------|--------------|--------|
| Alytaus l-d Nykštukas | Nykštukų g. 1 | A. Juozapavičiaus g. 44 | ❌ FABRICATED |
| Alytaus l-d Obelėlė | Obelų g. 1 | Šaltinių g. 3 | ❌ FABRICATED |
| Alytaus l-d Pasaka | Pasakų g. 1 | Vingio g. 7a | ❌ FABRICATED |
| Alytaus l-d Putinėlis | Kalnėnų g. 2 | Putinų g. 18 | ❌ WRONG |
| Alytaus l-d Pušynėlis | Pušyno g. 1 | Margio g. 1 | ❌ FABRICATED |
| Alytaus l-d Saulutė | Saulės g. 1 | Naujoji g. 32 | ❌ FABRICATED |
| Alytaus l-d Vyturėlis | Vyturių g. 1 | Miklusėnų g. 38a | ❌ FABRICATED |
| Alytaus l-d Šaltinėlis | Šaltinių g. 1 | Šaltinių g. 26 | ⚠️ WRONG NUMBER |

**Pattern detected:** Addresses are generated by deriving street names from kindergarten names (Nykštukas → Nykštukų g., Pasaka → Pasakų g.) and using house number 1. This is a strong indicator of fabrication.

**78 kindergartens have vague addresses** (just city name like "Kaunas" with no street), including:
- 30+ Kaunas kindergartens with just "Kaunas" as address
- 4 Jonava kindergartens with just "Jonava"
- Several Vilnius kindergartens with just district names ("Antakalnis, Vilnius")

Real Alytus landline phones follow pattern `+370 315 7XXXX` (area code 315). DB phones use random formats including mobile numbers assigned to public institutions (unlikely).

### 1.3b Phone Format Validation (all entities)
Phone format check across all 928 entities with phones:
- **Valid mobile:** 355 (38.3%)
- **Valid landline:** 487 (52.5%)
- **Wrong digit count:** 86 (9.3%) — landlines missing 1-2 subscriber digits
  - 30 aukles, 30 bureliai, 25 specialists, 0 kindergartens affected
  - Vilnius numbers (area code 5) need 7 subscriber digits but got only 5
  - Two-digit area codes (37, 41, 45, 46) need 6 subscriber digits but got only 5
- **1 special case:** Zigzag Travel aukle has `+370037407140` (double prefix)

Note: Kindergarten phones pass format validation (correct structure) but contain fabricated values (verified against real Alytus data — none match).

### 1.4 Websites — VERIFIED NON-FUNCTIONAL (99.7% fail)
Full HTTP HEAD check results across all entities with websites:

| Entity Type | PASS | FAIL | Total | Pass Rate |
|-------------|------|------|-------|-----------|
| Kindergarten | 2 | 769 | 771 | 0.3% |
| Burelis | 48 | 59 | 107 | 44.9% |
| Specialist | 33 | 1 | 34 | 97.1% |
| **TOTAL** | **83** | **829** | **912** | **9.1%** |

- 764 out of 771 kindergartens have `*.lm.lt` websites — the pattern IS correct for Lithuanian institutions but almost all timeout
- Only 2 kindergarten websites actually resolved
- Bureliai are mixed: real organizations (Bricks4Kidz, Edulando) pass, generated domains fail
- Specialist websites are almost all real and functional (97%)
- Verified real websites differ from DB: e.g., real Gintarėlis is at `gintarelisvilnius.lt`, not `gintarelis.vilnius.lm.lt`
- Note: `.lm.lt` government-hosted sites may block automated HEAD requests, but cross-checking with known real sites shows many URLs are simply fabricated
